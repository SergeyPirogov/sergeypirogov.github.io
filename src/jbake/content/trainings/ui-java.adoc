== UI Test Automation з Java - Від основ до CI/CD

Комплексний тренінг з автоматизації тестування UI за допомогою Java та Selenide.
Ви навчитесь створювати масштабовані проекти тестування UI з нуля до розгортування у CI/CD pipeline.

=== Для кого цей тренінг?

* Початківці в автоматизації UI тестування
* QA інженери, які хочуть навчитися Java та Selenide
* Java розробники, які переходять на тестування
* Тестувальники без досвіду програмування на Java

=== Що ви отримаєте?

* **Основи Java** - синтаксис, структури данних, ООП
* **Selenide фреймворк** - спрощена робота з Selenium
* **Пошук елементів** - локатори, стратегії, best practices
* **Page Object Model** - структурування тестового коду
* **Allure звітування** - красивіші звіти про результати
* **Jenkins CI/CD** - запуск тестів у pipeline та автоматизація

=== Які технології ми використовуємо?

* *Java 8* - мова програмування
* *Selenide* - фреймворк для UI тестування
* *Gradle* - систем білдування
* *Allure* - фреймворк звітування
* *Jenkins* - серверний контроль та CI/CD

=== Формат навчання

Інтерактивні онлайн лекції з практичними завданнями.
Ви напишете реальні UI тести разом із ментором протягом кожної лекції.
