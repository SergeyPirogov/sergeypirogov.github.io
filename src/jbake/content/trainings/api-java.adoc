== REST API Test Automation з Java - Від основ до CI/CD

Комплексний тренінг з автоматизації тестування REST API за допомогою Java.
Ви навчитесь створювати масштабовані проекти тестування REST API з нуля до розгортування у CI/CD pipeline.

=== Для кого цей тренінг?

* Початківці в автоматизації API тестування
* QA інженери, які хочуть навчитися REST API тестуванню
* Java розробники, які переходять на автоматизоване тестування
* Тестувальники без досвіду програмування на Java

=== Що ви отримаєте?

* **Розуміння REST API** - архітектура, HTTP методи, статус-коди
* **Rest Assured Framework** - написання масштабованих API тестів
* **Основи Java** - синтаксис, структури данних, ООП
* **Інтеграція з Gradle** - управління залежностями та структура проекту
* **Allure звітування** - красивіші звіти про результати тестів
* **Jenkins CI/CD** - запуск тестів у pipeline та автоматизація

=== Які технології ми використовуємо?

* *Java 8* - мова програмування
* *Rest Assured* - фреймворк для API тестування
* *Gradle* - систем білдування
* *Allure* - фреймворк звітування
* *Jenkins* - серверний контроль та CI/CD

=== Формат навчання

Інтерактивні онлайн лекції з практичними завданнями.
Ви напишете реальні API тести разом із ментором протягом кожної лекції.
