== UI Test Automation з Python - Від основ до CI/CD

Комплексний тренінг з автоматизації тестування UI за допомогою Python.
Ви навчитесь створювати масштабовані проекти тестування UI з нуля до розгортування у CI/CD pipeline.

=== Для кого цей тренінг?

* Початківці в автоматизації UI тестування
* QA інженери, які хочуть навчитися Python та Selenium/Playwright
* Python розробники, які переходять на тестування
* Тестувальники без досвіду програмування на Python

=== Що ви отримаєте?

* **Основи Python** - синтаксис, структури данних, ООП
* **Selenium та Playwright** - обидва фреймворки для UI тестування
* **Пошук елементів** - локатори, стратегії, best practices
* **Page Object Model** - структурування тестового коду
* **Allure та Report Portal** - красивіші звіти про результати
* **Jenkins CI/CD** - запуск тестів у pipeline та автоматизація

=== Які технології ми використовуємо?

* *Python 3* - мова програмування
* *Selenium* - класичний фреймворк для UI тестування
* *Playwright* - современний фреймворк для браузерної автоматизації
* *pytest* - фреймворк для тестування
* *Allure* - фреймворк звітування
* *Report Portal* - система управління звітами

=== Формат навчання

Інтерактивні онлайн лекції з практичними завданнями.
Ви напишете реальні UI тести разом із ментором протягом кожної лекції.
