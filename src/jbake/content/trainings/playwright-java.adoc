== Playwright Java UI Automation - Від Selenium до Современного Фреймворка

Комплексний тренінг з автоматизації тестування UI за допомогою Playwright Java.
Ви навчитесь створювати масштабовані проекти тестування UI з нуля до розгортування у CI/CD pipeline.

=== Для кого цей тренінг?

* Тестувальники, які працювали з Selenium/Selenide
* QA інженери, які хочуть спробувати Playwright
* Java розробники, які займаються тестуванням UI
* Тести писери, які шукають современного інструменту

=== Що ви отримаєте?

* **Переваги Playwright** - порівняння з Selenium, архітектура, можливості
* **Написання стійких тестів** - best practices, стратегії пошуку елементів
* **Основи Playwright Java** - встановлення, налаштування, перші тести
* **Page Object Model** - структурування тестового коду
* **GitHub/GitHub Actions CI/CD** - запуск тестів у pipeline
* **Налагодження та оптимізація** - вирішення проблем, прискорення тестів

=== Які технології ми використовуємо?

* *Java 17* - мова програмування
* *Playwright Java* - фреймворк для браузерної автоматизації
* *Gradle / Maven* - системи білдування
* *GitHub / GitHub Actions* - контроль версій та CI/CD
* *JUnit* - фреймворк для тестування

=== Формат навчання

Інтерактивні онлайн лекції з практичними завданнями.
Ви напишете реальні UI тести разом із ментором протягом кожної лекції.
