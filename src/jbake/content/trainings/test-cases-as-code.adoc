== Test Cases as Code - Управління Тестами як Кодом

Тренінг з управління тестами як кодом за допомогою Java та TestRail.
Ви навчитесь створювати проекти, де тести та їхня документація існують в одному місці.

=== Для кого цей тренінг?

* QA інженери, які хочуть впровадити Test Cases as Code підхід
* Тестувальники, які працюють з TestRail
* Java розробники, які займаються тестуванням
* Команди, які хочуть автоматизувати управління тестами

=== Що ви отримаєте?

* **Розуміння Test Cases as Code** - принципи та переваги підходу
* **Структура проекту** - як організувати код та документацію
* **Синхронізація з TestRail** - автоматичне оновлення тест кейсів
* **Мета-дані тестів** - теги, пріоритети, категорії
* **Генерація звітів** - автоматичні звіти про покриття
* **Масштабування** - управління великою кількістю тестів

=== Які технології ми використовуємо?

* *Java 8* - мова програмування
* *Gradle* - систем білдування
* *TestRail* - система управління тестами
* *IntelliJ IDEA* - IDE для розробки
* *Maven / Gradle плагіни* - для інтеграції з TestRail
