== GraphQL API Test Automation з Java - Від основ до CI/CD

Тренінг з автоматизації тестування GraphQL API за допомогою Java.
Ви навчитесь створювати масштабовані проекти тестування GraphQL сервісів з нуля до розгортування у CI/CD.

=== Для кого цей тренінг?

* Початківці в тестуванні GraphQL API
* QA інженери, які хочуть навчитися GraphQL
* Java розробники, які працюють з GraphQL
* Тестувальники, які шукають альтернативу REST API

=== Що ви отримаєте?

* **Розуміння GraphQL** - відмінності від REST, структура запитів, мутації
* **Написання GraphQL запитів** - синтаксис, змінні, фрагменти
* **Rest Assured для GraphQL** - відправка запитів та обробка відповідей
* **Валідація відповідей** - перевірка структури та содержимого
* **Allure звітування** - красивіші звіти про результати
* **CI/CD інтеграція** - запуск тестів у Jenkins pipeline

=== Які технології ми використовуємо?

* *Java 8/11* - мова програмування
* *Rest Assured* - фреймворк для API тестування
* *GraphQL* - мова запитів для API
* *Gradle* - систем білдування
* *Allure* - фреймворк звітування
* *Jenkins* - серверний контроль та CI/CD

=== Формат навчання

Інтерактивні онлайн лекції з практичними завданнями.
Ви напишете реальні GraphQL тести разом із ментором протягом кожної лекції.
