Про автора

QA Automation Engineer · Спікер · Open Source Contributor

Досвідчений QA Automation спеціаліст з 10-річним досвідом створення проектів з нуля. Перевірений досвід у розробці та впровадженні наскрізних рішень автоматизації. Мої ключові компетенції включають автоматизацію тестування, планування тестування, Java, Selenium, Python, JS/TypeScript, SQL, Jenkins, Git/GitHub, Linux, Docker і хмарні технології. Значний досвід роботи з процесами CI/CD. Ефективна співпраця з міжфункціональними командами для узгодження стратегій тестування та забезпечення високоякісного постачання програмного забезпечення. Постійне вдосконалення та оновлення останніх галузевих тенденцій для оптимізації процесів тестування та підвищення ефективності.

Окрім моєї професійної ролі, я також зробив значний внесок у розробку open source. Я розробив бібліотеку Python для керування двійковими файлами Selenium WebDriver, яку офіційно схвалила команда Selenium.dev, і створив версію Python для проекту Testcontainers, який був прийнятий до офіційної організації Testcontainers GitHub.

Як активний спікер і член програмного комітету конференцій із забезпечення якості, я прагну підтримувати професійний ріст спільноти з контролю якості. Я став переможцем Ukraine IT Award 2018 у номінації «Забезпечення якості» за активну участь у розвитку QA спільноти в Україні.
