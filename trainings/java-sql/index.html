Java + SQL для Тестування - Робота з Базами Данних

Тренінг з роботи з базами данних за допомогою Java. Ви навчитесь писати SQL запити та інтегрувати їх у автоматизовані тести.

Для кого цей тренінг?

  • QA інженери, які хочуть працювати з базами данних

  • Тестувальники, які хочуть валідувати дані у базі

  • Java розробники, які переходять на тестування

  • Тести писери, які потребують прямого доступу до БД

Що ви отримаєте?

  • Основи SQL - SELECT, INSERT, UPDATE, DELETE запити

  • Робота з JDBC - підключення до баз данних

  • DbUtils бібліотека - спрощена робота з результатами

  • JDBI фреймворк - зручна абстракція над JDBC

  • Сценарії тестування - перевірка даних у БД після тестів

  • Очищення даних - підготовка тестового навколишнього середовища

Які технології ми використовуємо?

  • Java 8 - мова програмування

  • JDBC - API для роботи з БД

  • DbUtils - утиліти для роботи з результатами

  • JDBI - SQL об’єктний маппер

  • Gradle - систем білдування

  • PostgreSQL - база даних

Формат навчання

Інтерактивні онлайн лекції з практичними завданнями. Ви напишете SQL запити та інтегруватимете їх у тести разом із ментором.
